# project 1: calculator

## Project Goal
目标: 掌握函数、异常处理、用户输入
This project is created as part of a Python learning journey.
The goal is to practice:
- Basic Python syntax
- Function design
- Command-line interaction
- Error handling


## Features
- 支持加减乘除、幂运算
- 历史记录功能
- 异常处理（除零、无效输入）
- 命令行界面

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles invalid input and division by zero


## Key point

✓ 字符串格式化
